# 🐍 Python Beginner Projects Collection

This repository contains 7 simple beginner-friendly Python projects.  
Each project is console-based and helps in building strong programming fundamentals.

---

## 📌 Projects Included

### 1️⃣ Expense Tracker 💰
Track your daily expenses and manage simple budgets.

### 2️⃣ Rock Paper Scissors 🎮
Classic game against computer using random module.

### 3️⃣ ATM Banking System 🏦
Simulates basic ATM operations like balance check, deposit, and withdrawal.

### 4️⃣ Calculator ➗
Performs basic arithmetic operations.

### 5️⃣ Number Guessing Game 🎯
Guess the randomly generated number with hints.

### 6️⃣ Snake Water Gun 🐍💧🔫
Fun variation of Rock Paper Scissors.

### 7️⃣ Student Management System 🎓
Manage student records like add, view, and update details.
